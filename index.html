Learn By Scraping

A guide and a tool to learn to scraping by scraping

Introduction

As guides are often long stretches of text, they can come off as daunting - and as they are generally not accompanied by practice, they likely do not inspire confidence within the reader in their ability to actually implement the guide's instructions. As such, we have created this site which functions as both a guide to teach you and a tool to give you practical experience.

Language Specific Details

In this section, we will cover language specific details, these will include:

  • Libraries to use
  • How to use them

but will not include commonalities, such as how to form css selectors. Rather, those will have their own dedicated sections.

In this section, we will add sub-sections for the following languages:

  • Python
  • *Nix shells

but PRs for language additions are welcome.

The following language-specific sub-sections will show you how to send requests in your language as well as how to parse the results. However, some things which may be mentioned, such as css selectors, xpaths, and regex will NOT be elaborated in here as they will be given their own sections since they are a commanality between languages.

Python

For scraping with python, we will be using the following libraries:

  • httpx
    • Use: To create network requests to desired URI's
  • beautifulsoup
    • Use: To Parse the request body into a pythonic object

These can be downloaded using the following commands:

  • python -m pip install httpx

  • python -m pip install beautifulsoup4

To import httpx and beautifulsoup - you must import them as follows:

import httpx
from bs4 import BeautifulSoup
